Over de toepassingslaag

De toepassings laag is het deel van de computer waar gebruikers het meeste baat bij hebben Dit is de laag met alle programmas en apps die je gebruikt, zoals Google, Spotify, of Adobe. Dit is waar de meeste mensen een computer voor aanschaffen. De toepassingslaag maakt gebruik van de logische en fysieke lagen om te werken, zonder dat je daar echt iets van merkt.

Apps zijn programmas waarmee de gebruiker iets op de computer kan doen. Denk aan een app voor social media, om fotoâ€™s te bewerken of video games. Apps werken door te communiceren met het besturingssysteem en de hardware. Apps zijn er in allerlei vormen en kunnen op je telefoon, computer, of tablet draaien. Sommige apps werken alleen als je internet hebt, en andere kun je offline gebruiken. Sommige Apps werken ook niet op alle soorten OS. Op onze schoollaptop kunnen we bijvoorbeeld geen vs code gebruiken, omdat dat niet kan op ChromeOS.

API staat voor Application Programming Interface en is een manier waarop verschillende programmas en apps met elkaar kunnen samenwerken. Een API zorgt ervoor dat een app bepaalde informatie of functies van een ander programma kan gebruiken. Bijvoorbeeld als een weer-app de weersvoorspelling laat zien, gebruikt die waarschijnlijk een API om data op te halen van een weerwebsite. Een API zorgt er dus voor dat apps niet zelf alles hoeven te weten, maar toch toegang hebben tot handige data of functies van andere programmas of servers.
